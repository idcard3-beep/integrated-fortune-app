# 상담 전문가 사진 폴더

이 폴더는 상담 전문가 프로필 사진을 저장하는 곳입니다.

## 파일명 규칙

사진 파일은 다음과 같은 이름으로 저장해야 합니다:

- slide1.jpg
- slide2.jpg
- slide3.jpg
- ...
- slide50.jpg

## 사용 방법

1. 전문가 사진을 위 파일명 규칙에 맞춰 이 폴더에 업로드합니다.
2. 0103_couns_expert.html 페이지에서 전문가 정보를 추가/수정할 때 파일명을 입력합니다.
   - 예: "slide1.jpg"
3. 시스템이 자동으로 `/uploads/images/couns/` 경로와 결합하여 이미지를 표시합니다.

## 권장 이미지 사양

- 해상도: 최소 300x360px (권장 400x480px)
- 비율: 5:6 (세로형)
- 포맷: JPG, PNG
- 파일 크기: 최대 500KB (최적화 권장)
